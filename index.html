Введение

Visual Studio Code - это мощный редактор кода с открытым исходным кодом, разработанный Microsoft. Эта шпаргалка содержит наиболее полезные комбинации клавиш и советы для macOS, которые помогут вам повысить продуктивность работы.

Примечание: В VS Code используются следующие модификаторы клавиш для macOS:

  • ⌘ (Command)
  • ⌥ (Option/Alt)
  • ⇧ (Shift)
  • ⌃ (Control)

Настройка сочетаний клавиш

VS Code позволяет полностью настраивать сочетания клавиш под себя. Вот как это сделать:

Открыть редактор сочетаний клавиш

Нажмите ⌘ + K затем ⌘ + S или выберите Code → Preferences → Keyboard Shortcuts.

Поиск сочетаний клавиш

В редакторе сочетаний клавиш вы можете:

  • Искать по названию команды
  • Искать по текущему сочетанию клавиш
  • Искать по категории

Изменение сочетания клавиш

Чтобы изменить сочетание клавиш:

  1. Найдите команду, сочетание для которой вы хотите изменить
  2. Нажмите на значок "+" справа от команды
  3. Нажмите желаемую комбинацию клавиш
  4. Нажмите Enter для сохранения

Ручное редактирование файла keybindings.json

Вы также можете напрямую редактировать файл конфигурации сочетаний клавиш:

  1. Откройте редактор сочетаний клавиш (⌘ + K затем ⌘ + S)
  2. Нажмите на значок "Open Keyboard Shortcuts (JSON)" в правом верхнем углу
  3. Добавьте свои сочетания клавиш в формате JSON

Пример записи в keybindings.json:

[
  {
    "key": "cmd+k e",
    "command": "workbench.files.action.focusOpenEditorsView",
    "when": "workbench.explorer.openEditorsView.active"
  },
  {
    "key": "cmd+k c",
    "command": "workbench.files.action.compareWithClipboard"
  },
  {
    "key": "alt+cmd+c",
    "command": "editor.action.commentLine",
    "when": "editorTextFocus && !editorReadonly"
  }
]

Ключевые компоненты настройки сочетаний клавиш

  • key: комбинация клавиш
  • command: команда, которую нужно выполнить
  • when: условие, при котором действует сочетание (опционально)

Условные сочетания клавиш

Вы можете создавать сочетания клавиш, которые работают только в определенных контекстах, используя условие when:

  • editorFocus: когда редактор в фокусе
  • terminalFocus: когда терминал в фокусе
  • explorerViewletVisible: когда explorer виден
  • searchViewletVisible: когда поиск виден
  • и многие другие...

Установка расширений через командную строку

Вы можете устанавливать расширения напрямую через терминал с помощью команды code:

code --install-extension publisher.extension-name

Пример:

code --install-extension dbaeumer.vscode-eslint

Совет: Чтобы использовать команду code в терминале, вам нужно сначала установить CLI. Откройте командную палитру (⇧ + ⌘ + P) и выполните команду "Shell Command: Install 'code' command in PATH".

Советы и приемы для повышения продуктивности

Настройки VS Code для macOS

Некоторые особенности настройки VS Code для работы на macOS:

  • Системные сочетания клавиш: Некоторые сочетания клавиш VS Code могут конфликтовать с системными. Проверьте в Системных настройках → Клавиатура → Сочетания клавиш.
  • Terminal Shell: Настройте стандартный shell в settings.json: 
    "terminal.integrated.defaultProfile.osx": "zsh"
  • Font Ligatures: Включите лигатуры для лучшей читабельности кода: 
    "editor.fontFamily": "Fira Code", 
"editor.fontLigatures": true

Настройки Touch Bar

VS Code поддерживает настройку Touch Bar на MacBook Pro. Вы можете добавить в settings.json:

"keyboard.touchbar.enabled": true,
"keyboard.touchbar.ignored": []

Сниппеты кода

Создание собственных сниппетов может значительно ускорить вашу работу:

  1. Откройте командную палитру (⇧ + ⌘ + P)
  2. Введите "Snippets: Configure User Snippets"
  3. Выберите язык, для которого хотите создать сниппет

Пример сниппета для JavaScript:

{
  "Console log": {
    "prefix": "cl",
    "body": [
      "console.log($1);"
    ],
    "description": "Console log"
  }
}

Многооконный режим работы

VS Code поддерживает работу с несколькими окнами и группами редакторов:

  • ⌘ + \\ - Разделить редактор
  • ⌘ + 1/2/3 - Фокус на 1-й/2-й/3-й группе редакторов
  • ⌃ + ⌘ + N - Открыть новое окно VS Code

Встроенные инструменты повышения производительности

  • Zen Mode: ⌘ + K затем Z - режим без отвлекающих элементов
  • Breadcrumbs: Навигация по структуре файла (включить в настройках)
  • Find All References: ⇧ + F12 - найти все использования символа
  • Linked Editing: "editor.linkedEditing": true - автоматически переименовывает HTML теги

Git интеграция

VS Code имеет мощную интеграцию с Git:

  • ⌃ + ⇧ + G - Открыть панель Git
  • ⌘ + K ⌘ + G - Status панель Git
  • Отображение изменений в редакторе (gutter)
  • Встроенный инструмент разрешения конфликтов

Оптимизация для Mac с Apple Silicon

Начиная с VS Code версии 1.54, добавлена нативная поддержка для Mac с процессорами Apple Silicon (M1/M2/M3). Оптимизированная версия обеспечивает:

  • Повышенную производительность
  • Лучшее время автономной работы
  • Больше совместимых расширений

Убедитесь, что у вас установлена версия для Apple Silicon, а не запущенная через Rosetta 2 x86 версия.
